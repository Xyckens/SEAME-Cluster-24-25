\doxysubsection{Detailed Description}
libcamera image pixel format 

The \doxylink{classlibcamera_1_1PixelFormat}{Pixel\+Format} type describes the format of images in the public libcamera API. It stores a Four\+CC value as a 32-\/bit unsigned integer and a modifier. The Four\+CC and modifier values are defined in the Linux kernel DRM/\+KMS API (see \doxylink{drm__fourcc_8h_source}{linux/drm\+\_\+fourcc.\+h}). Constant expressions for all pixel formats supported by libcamera are available in libcamera/formats.\+h.
