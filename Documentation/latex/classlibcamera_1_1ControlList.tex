\doxysubsection{Detailed Description}
Associate a list of \doxylink{classlibcamera_1_1ControlId}{Control\+Id} with their values for an object. 

The \doxylink{classlibcamera_1_1ControlList}{Control\+List} class stores values of controls exposed by an object. The lists returned by the \doxylink{classlibcamera_1_1Request_ae9327d49f60d15c34c90bb3d2f4cba10}{Request\+::controls()} and \doxylink{classlibcamera_1_1Request_ac620539cc50a28bd9ff9f8f697eb86c3}{Request\+::metadata()} functions refer to the camera that the request belongs to.

\doxylink{classlibcamera_1_1Control}{Control} lists are constructed with a map of all the controls supported by their object, and an optional \doxylink{classlibcamera_1_1ControlValidator}{Control\+Validator} to further validate the controls.

Merging two control lists copies elements from the {\itshape source} and inserts them in \texorpdfstring{$\ast$}{*}this. If the {\itshape source} contains elements whose key is already present in \texorpdfstring{$\ast$}{*}this, then those elements are only overwritten if {\itshape policy} is \doxylink{classlibcamera_1_1ControlList_a61c8b9a57df34e40716d3664d24df80da213fc6f9c23a879020bc94b79cc51d62}{Merge\+Policy\+::\+Overwrite\+Existing}.
