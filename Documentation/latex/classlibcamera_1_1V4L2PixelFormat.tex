\doxysubsection{Detailed Description}
V4\+L2 pixel format Four\+CC wrapper. 

The \doxylink{classlibcamera_1_1V4L2PixelFormat}{V4\+L2\+Pixel\+Format} class describes the pixel format of a V4\+L2 buffer. It wraps the V4\+L2 numerical Four\+CC, and shall be used in all APIs that deal with V4\+L2 pixel formats. Its purpose is to prevent unintentional confusion of V4\+L2 and DRM Four\+CCs in code by catching implicit conversion attempts at compile time.

To achieve this goal, construction of a \doxylink{classlibcamera_1_1V4L2PixelFormat}{V4\+L2\+Pixel\+Format} from an integer value is explicit. To retrieve the integer value of a \doxylink{classlibcamera_1_1V4L2PixelFormat}{V4\+L2\+Pixel\+Format}, both the explicit value() and implicit uint32\+\_\+t conversion operators may be used.
