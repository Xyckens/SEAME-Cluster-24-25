{\footnotesize\ttfamily bool libcamera\+::\+IPAModule\+::load (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Load the IPA implementation factory from the shared object. 

The IPA module shared object implements an \doxylink{classlibcamera_1_1IPAInterface}{IPAInterface} object to be used by pipeline handlers. This function loads the factory function from the shared object. Later, \doxylink{classlibcamera_1_1IPAModule_af70448b68ca0f671376b4ca93e402e79}{create\+Interface()} can be called to instantiate the \doxylink{classlibcamera_1_1IPAInterface}{IPAInterface}.

This function only needs to be called successfully once, after which \doxylink{classlibcamera_1_1IPAModule_af70448b68ca0f671376b4ca93e402e79}{create\+Interface()} can be called as many times as \doxylink{classlibcamera_1_1IPAInterface}{IPAInterface} instances are needed.

Calling this function on an invalid module (as returned by \doxylink{classlibcamera_1_1IPAModule_af14705dc1eebb33518e8c3aacf3b7957}{is\+Valid()}) is an error.

\begin{DoxyReturn}{Returns}
True if load was successful, or already loaded, and false otherwise 
\end{DoxyReturn}

Retrieve the IPA module signature. 

The IPA module signature is stored alongside the IPA module in a file with a \textquotesingle{}.sign\textquotesingle{} suffix, and is loaded when the \doxylink{classlibcamera_1_1IPAModule}{IPAModule} instance is created. This function returns the signature without verifying it. If the signature is missing, the returned vector will be empty.
