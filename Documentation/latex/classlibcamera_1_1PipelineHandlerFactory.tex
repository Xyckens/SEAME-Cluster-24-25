\begin{DoxyTemplParams}{Template Parameters}

To facilitate discovery and instantiation of \doxylink{classlibcamera_1_1PipelineHandler}{Pipeline\+Handler} classes, the \doxylink{classlibcamera_1_1PipelineHandlerFactory}{Pipeline\+Handler\+Factory} class implements auto-\/registration of pipeline handlers. Each \doxylink{classlibcamera_1_1PipelineHandler}{Pipeline\+Handler} subclass shall register itself using the \doxylink{pipeline__handler_8h_a8dae35473d8718ef287a93c2092c47bb}{REGISTER\+\_\+\+PIPELINE\+\_\+\+HANDLER()} macro, which will create a corresponding instance of a \doxylink{classlibcamera_1_1PipelineHandlerFactory}{Pipeline\+Handler\+Factory} and register it with the static list of factories.
