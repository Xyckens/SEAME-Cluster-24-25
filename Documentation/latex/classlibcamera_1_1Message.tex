Reserve and register a custom user-\/defined message type. 

Custom message types use values starting at \doxylink{classlibcamera_1_1Message_a7ae15d74f028344ba827ea8ba4d77fa5ad7c9da4e25599a32fe48308e6da49cf1}{Message\+::\+User\+Message}. Assigning custom types manually may lead to accidental duplicated types. To avoid this problem, this function reserves and returns the next available user-\/defined message type.

The recommended way to use this function is to subclass \doxylink{classlibcamera_1_1Message}{Message} and provide a static accessor for the custom message type.
