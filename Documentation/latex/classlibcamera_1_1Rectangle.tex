\doxysubsection{Detailed Description}
Describe a rectangle\textquotesingle{}s position and dimensions. 

Rectangles are used to identify an area of an image. They are specified by the coordinates of top-\/left corner and their horizontal and vertical size. By convention, the top-\/left corner is defined as the corner with the lowest x and y coordinates, regardless of the origin and direction of the axes.

The measure unit of the rectangle coordinates and size, as well as the reference point from which the \doxylink{classlibcamera_1_1Rectangle_af2016608aee722b737b85694a1d1d52c}{Rectangle\+::x} and \doxylink{classlibcamera_1_1Rectangle_a0e9e38aa81a1e812e003c29cc1ad1b02}{Rectangle\+::y} displacements refers to, are defined by the context were rectangle is used.

The \doxylink{classlibcamera_1_1Rectangle}{Rectangle} is modified so that it does not exceed the given {\itshape boundary}. This process involves translating the \doxylink{classlibcamera_1_1Rectangle}{Rectangle} if any of its edges lie beyond {\itshape boundary}, so that those edges then lie along the boundary instead.

If either width or height are larger than {\itshape boundary}, then the returned \doxylink{classlibcamera_1_1Rectangle}{Rectangle} is clipped to be no larger. But other than this, the \doxylink{classlibcamera_1_1Rectangle}{Rectangle} is not clipped or reduced in size, merely translated.

Note that this is not a conventional \doxylink{classlibcamera_1_1Rectangle}{Rectangle} intersection function which is provided by \doxylink{classlibcamera_1_1Rectangle_af6ab6db441d6ebb0c3032c0b26df243e}{bounded\+To()}.
