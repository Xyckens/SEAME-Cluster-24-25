This function is called from the post-\/processor\textquotesingle{}s thread whenever a camera stream has finished post processing. The corresponding entry is dropped from the descriptor\textquotesingle{}s pending\+Streams\+To\+Process\+\_\+ map.

If the pending\+Streams\+To\+Process\+\_\+ map is then empty, all streams requiring to be generated from post-\/processing have been completed. Mark the descriptor as complete using complete\+Descriptor() in that case.
