\doxysubsection{Detailed Description}
\doxylink{classlibcamera_1_1Control}{Control} static metadata. 

The \doxylink{classlibcamera_1_1ControlId}{Control\+Id} class stores a control ID, name and data type. It provides unique identification of a control, but without support for compile-\/time type deduction that the derived template \doxylink{classlibcamera_1_1Control}{Control} class supports. See the \doxylink{classlibcamera_1_1Control}{Control} class for more information.
