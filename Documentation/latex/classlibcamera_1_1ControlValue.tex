Get the control value. 

This function returns the contained value as an instance of {\itshape T}. If the \doxylink{classlibcamera_1_1ControlValue}{Control\+Value} instance stores a single value, the type {\itshape T} shall match the stored value \doxylink{classlibcamera_1_1ControlValue_a92983704d12eb6a41e8971835712e350}{type()}. If the instance stores an array of values, the type {\itshape T} should be equal to Span$<$const R$>$, and the type {\itshape R} shall match the stored value \doxylink{classlibcamera_1_1ControlValue_a92983704d12eb6a41e8971835712e350}{type()}. The behaviour is undefined otherwise.

Note that a \doxylink{classlibcamera_1_1ControlValue}{Control\+Value} instance that stores a non-\/array value is not equivalent to an instance that stores an array value containing a single element. The latter shall be accessed through a Span$<$const R$>$ type, while the former shall be accessed through a type {\itshape T} corresponding to \doxylink{classlibcamera_1_1ControlValue_a92983704d12eb6a41e8971835712e350}{type()}.

Otherwise the instance stores a single value of type {\itshape T}.
