\doxysubsection{Detailed Description}
A helper class for parsing a YAML file. 

The \doxylink{classlibcamera_1_1YamlParser}{Yaml\+Parser} class provides an easy interface to parse the contents of a YAML file into a tree of \doxylink{classlibcamera_1_1YamlObject}{Yaml\+Object} instances.
