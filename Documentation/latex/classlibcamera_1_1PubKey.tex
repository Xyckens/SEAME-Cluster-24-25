\doxysubsection{Detailed Description}
Public key wrapper for signature verification. 

The \doxylink{classlibcamera_1_1PubKey}{Pub\+Key} class wraps a public key and implements signature verification. It only supports RSA keys and the RSA-\/\+SHA256 signature algorithm.
