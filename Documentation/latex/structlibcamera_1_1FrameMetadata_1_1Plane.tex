\doxysubsection{Detailed Description}
Per-\/plane frame metadata. 

Frames are stored in memory in one or multiple planes. The \doxylink{structlibcamera_1_1FrameMetadata_1_1Plane}{Frame\+Metadata\+::\+Plane} structure stores per-\/plane metadata.
