\doxysubsection{Detailed Description}
Describe the ImgU requested configuration. 

The ImgU unit processes images through several components, which have to be properly configured inspecting the input image size and the desired output sizes. This structure collects the ImgU input configuration and the requested main output and viewfinder configurations.
