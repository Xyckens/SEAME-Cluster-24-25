\doxysubsection{Detailed Description}
Group information for per-\/stream buffer of \doxylink{classCamera3RequestDescriptor}{Camera3\+Request\+Descriptor}. 

A capture request placed to the libcamera HAL can contain multiple streams. Each stream will have an associated buffer to be filled. \doxylink{structCamera3RequestDescriptor_1_1StreamBuffer}{Stream\+Buffer} tracks this buffer with contextual information which aids in the stream\textquotesingle{}s generation. The generation of the stream will depend on its type (refer to the Camera\+Stream\+::\+Type documentation).
