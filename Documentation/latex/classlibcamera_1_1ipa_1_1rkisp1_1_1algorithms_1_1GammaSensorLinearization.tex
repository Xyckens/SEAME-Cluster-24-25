\doxysubsection{Detailed Description}
Rk\+ISP1 Gamma Sensor Linearization control. 

This algorithm linearizes the sensor output to compensate the sensor non-\/linearities by applying piecewise linear functions to the red, green and blue channels.

The curves are specified in the tuning data and defined using 17 points.


\begin{DoxyItemize}
\item The X coordinates are expressed using 16 intervals, with the first point at X coordinate 0. Each interval is expressed as a 2-\/bit value DX (from GAMMA\+\_\+\+DX\+\_\+1 to GAMMA\+\_\+\+DX\+\_\+16), stored in the RKISP1\+\_\+\+CIF\+\_\+\+ISP\+\_\+\+GAMMA\+\_\+\+DX\+\_\+\+LO and RKISP1\+\_\+\+CIF\+\_\+\+ISP\+\_\+\+GAMMA\+\_\+\+DX\+\_\+\+HI registers. The real interval is equal to $2^{dx+4}$. X coordinates are shared between the red, green and blue curves.
\item The Y coordinates are specified as 17 values separately for the red, green and blue channels, with a 12-\/bit resolution. Each value must be in the \mbox{[}-\/2048, 2047\mbox{]} range compared to the previous value. 
\end{DoxyItemize}
