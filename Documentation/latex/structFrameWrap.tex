\doxysubsection{Detailed Description}
An internal wrapper to track the relation between Frame\+Buffer and Gst\+Memory(s) 

This wrapper maintains a count of the outstanding Gst\+Memory (there may be multiple Gst\+Memory per Frame\+Buffer), and give back the Frame\+Buffer to the allocator pool when all memory objects have returned.
