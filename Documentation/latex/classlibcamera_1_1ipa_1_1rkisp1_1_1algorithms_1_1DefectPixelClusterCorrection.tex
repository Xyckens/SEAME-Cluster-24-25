\doxysubsection{Detailed Description}
Rk\+ISP1 Defect Pixel Cluster Correction control. 

Depending of the sensor quality, some pixels can be defective and then appear significantly brighter or darker than the other pixels.

The Defect Pixel Cluster Correction algorithms is responsible to minimize the impact of the pixels. This can be done with algorithms applied at run time (on-\/the-\/fly method) or with a table of defective pixels. Only the first method is supported for the moment.
