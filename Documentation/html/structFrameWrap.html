Detailed Description

An internal wrapper to track the relation between FrameBuffer and GstMemory(s)

This wrapper maintains a count of the outstanding GstMemory (there may be multiple GstMemory per FrameBuffer), and give back the FrameBuffer to the allocator pool when all memory objects have returned.
