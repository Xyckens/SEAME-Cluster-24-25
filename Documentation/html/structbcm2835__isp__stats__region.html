Detailed Description

struct bcm2835_isp_stats_region - Region sums.

@counted: The number of 2x2 bayer tiles accumulated. @notcounted: The number of 2x2 bayer tiles not accumulated. @r_sum: Total sum of counted pixels in the red channel for a region. @g_sum: Total sum of counted pixels in the green channel for a region. @b_sum: Total sum of counted pixels in the blue channel for a region.
