struct ipu3_uapi_awb_config_s - AWB config

@rgbs_thr_gr: gr threshold value. @rgbs_thr_r: Red threshold value. @rgbs_thr_gb: gb threshold value. @rgbs_thr_b: Blue threshold value. @grid: &ipu3_uapi_grid_config, the default grid resolution is 16x16 cells.

The threshold is a saturation measure range [0, 8191], 8191 is default. Values over threshold may be optionally rejected for averaging.
