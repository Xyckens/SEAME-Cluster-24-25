Detailed Description

Group information for per-stream buffer of Camera3RequestDescriptor.

A capture request placed to the libcamera HAL can contain multiple streams. Each stream will have an associated buffer to be filled. StreamBuffer tracks this buffer with contextual information which aids in the stream's generation. The generation of the stream will depend on its type (refer to the CameraStream::Type documentation).
