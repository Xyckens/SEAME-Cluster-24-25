Detailed Description

struct v4l2_av1_loop_restoration - AV1 Loop Restauration as described in section 6.10.15 "Loop restoration params semantics" of the AV1 specification.

@flags: See V4L2_AV1_LOOP_RESTORATION_FLAG_{}. @lr_unit_shift: specifies if the luma restoration size should be halved. @lr_uv_shift: specifies if the chroma size should be half the luma size. @reserved: padding field. Should be zeroed by applications. @frame_restoration_type: specifies the type of restoration used for each plane. See enum v4l2_av1_frame_restoration_type. @loop_restoration_size: specifies the size of loop restoration units in units of samples in the current plane.
