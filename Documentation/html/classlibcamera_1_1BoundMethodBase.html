Invoke the bound method with packed arguments.

Parameters
[in]packPacked arguments
[in]deleteMethodTrue to delete this bound method instance when method invocation completes

The bound method stores its return value, if any, in the arguments pack. For direct and blocking invocations, this is performed synchronously, and the return value contained in the pack may be used. For queued invocations, the return value is stored at an undefined point of time and shall thus not be used by the caller.

Returns
True if the return value contained in the pack may be used by the caller, false otherwise
