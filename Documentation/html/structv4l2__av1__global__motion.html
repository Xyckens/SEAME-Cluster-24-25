Detailed Description

struct v4l2_av1_global_motion - AV1 Global Motion parameters as described in section 6.8.17 "Global motion params semantics" of the AV1 specification.

@flags: A bitfield containing the flags per reference frame. See V4L2_AV1_GLOBAL_MOTION_FLAG_{} @type: The type of global motion transform used. @params: this field has the same meaning as "gm_params" in the AV1 specification. @invalid: bitfield indicating whether the global motion params are invalid for a given reference frame. See section 7.11.3.6 Setup shear process and the variable "warpValid". Use V4L2_AV1_GLOBAL_MOTION_IS_INVALID(ref) to create a suitable mask. @reserved: padding field. Should be zeroed by applications.
