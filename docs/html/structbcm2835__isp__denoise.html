Detailed Description

struct bcm2835_isp_denoise - Denoise parameters set with the V4L2_CID_USER_BCM2835_ISP_DENOISE ctrl.

@enabled: Enable denoise. @constant: Fixed offset of the noise threshold. @slope: Slope of the noise threshold. @strength: Denoise strength between 0.0 (off) and 1.0 (maximum).
