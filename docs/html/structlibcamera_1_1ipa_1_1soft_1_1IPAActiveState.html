Detailed Description

The active state of the IPA algorithms.

The IPA is fed with the statistics generated from the latest frame processed. The statistics are then processed by the IPA algorithms to compute parameters required for the next frame capture and processing. The current state of the algorithms is reflected through the IPAActiveState to store the values most recently computed by the IPA algorithms.
