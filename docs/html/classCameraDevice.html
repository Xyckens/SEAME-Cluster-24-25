)

Handle post-processing completion of a stream in a capture request.

Parameters
[in]streamBufferThe StreamBuffer for which processing is complete
[in]statusStream post-processing status

This function is called from the post-processor's thread whenever a camera stream has finished post processing. The corresponding entry is dropped from the descriptor's pendingStreamsToProcess_ map.

If the pendingStreamsToProcess_ map is then empty, all streams requiring to be generated from post-processing have been completed. Mark the descriptor as complete using completeDescriptor() in that case.
