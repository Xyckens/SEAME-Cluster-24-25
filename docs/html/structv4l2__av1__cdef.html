Detailed Description

struct v4l2_av1_cdef - AV1 CDEF params semantics as described in section 6.10.14 "CDEF params semantics" of the AV1 specification

@damping_minus_3: controls the amount of damping in the deringing filter. @bits: specifies the number of bits needed to specify which CDEF filter to apply. @y_pri_strength: specifies the strength of the primary filter. @y_sec_strength: specifies the strength of the secondary filter. @uv_pri_strength: specifies the strength of the primary filter. @uv_sec_strength: specifies the strength of the secondary filter.
