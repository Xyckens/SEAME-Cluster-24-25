Detailed Description

struct v4l2_ctrl_av1_tile_group_entry - AV1 Tile Group entry

Represents a single AV1 tile inside an AV1 Tile Group. Note that MiRowStart, MiRowEnd, MiColStart and MiColEnd can be retrieved from struct v4l2_av1_tile_info in struct v4l2_ctrl_av1_frame using tile_row and tile_col. See section 6.10.1 "General tile group OBU semantics" for more details.

@tile_offset: offset from the OBU data, i.e. where the coded tile data actually starts. @tile_size: specifies the size in bytes of the coded tile. Equivalent to "TileSize" in the AV1 Specification. @tile_row: specifies the row of the current tile. Equivalent to "TileRow" in the AV1 Specification. @tile_col: specifies the col of the current tile. Equivalent to "TileCol" in the AV1 Specification.
