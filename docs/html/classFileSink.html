bool FileSink::processRequest ( libcamera::Request *  request)
overridevirtual
Parameters
[in]requestThe request

This function is called to instruct the sink to process a request. The sink may process the request synchronously or queue it for asynchronous processing.

When the request is processed synchronously, this function shall return true. The request shall not be accessed by the FrameSink after the function returns.

When the request is processed asynchronously, the FrameSink temporarily takes ownership of the request. The function shall return false, and the FrameSink shall emit the requestProcessed signal when the request processing completes. If the stop() function is called before the request processing completes, it shall release the request synchronously.

Returns
True if the request has been processed synchronously, false if processing has been queued
