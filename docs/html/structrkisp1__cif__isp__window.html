Detailed Description

struct rkisp1_cif_isp_window - measurement window.

Measurements are calculated per window inside the frame. This struct represents a window for a measurement.

@h_offs: the horizontal offset of the window from the left of the frame in pixels. @v_offs: the vertical offset of the window from the top of the frame in pixels. @h_size: the horizontal size of the window in pixels @v_size: the vertical size of the window in pixels.
