Detailed Description

struct bcm2835_isp_sharpen - Sharpen parameters set with the V4L2_CID_USER_BCM2835_ISP_SHARPEN ctrl.

@enabled: Enable sharpening. @threshold: Threshold at which to start sharpening pixels. @strength: Strength with which pixel sharpening increases. @limit: Limit to the amount of sharpening applied.
