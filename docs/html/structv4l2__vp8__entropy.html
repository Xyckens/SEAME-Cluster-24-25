Detailed Description

struct v4l2_vp8_entropy - VP8 update probabilities

@coeff_probs: coefficient probability update values. @y_mode_probs: luma intra-prediction probabilities. @uv_mode_probs: chroma intra-prediction probabilities. @mv_probs: mv decoding probability. @padding: padding field. Should be zeroed by applications.

This structure contains the update probabilities present in 'token_prob_update()' and 'mv_prob_update()' part of the frame header. See section '17.2. Probability Updates' of the VP8 specification for more details.
