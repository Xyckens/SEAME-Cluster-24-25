Detailed Description

struct v4l2_av1_tile_info - AV1 Tile info as defined in section 6.8.14 "Tile info semantics" of the AV1 specification.

@flags: see V4L2_AV1_TILE_INFO_FLAG_{} @context_update_tile_id: specifies which tile to use for the CDF update. @tile_rows: specifies the number of tiles down the frame. @tile_cols: specifies the number of tiles across the frame. @mi_col_starts: an array specifying the start column (in units of 4x4 luma samples) for each tile across the image. @mi_row_starts: an array specifying the start row (in units of 4x4 luma samples) for each tile down the image. @width_in_sbs_minus_1: specifies the width of a tile minus 1 in units of superblocks. @height_in_sbs_minus_1: specifies the height of a tile minus 1 in units of superblocks. @tile_size_bytes: specifies the number of bytes needed to code each tile size. @reserved: padding field. Should be zeroed by applications.
