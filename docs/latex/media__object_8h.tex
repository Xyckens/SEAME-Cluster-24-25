\doxysubsection{Detailed Description}
Provides a class hierarchy that represents the media objects exposed by the Linux kernel Media Controller APIs. 

The abstract Media\+Object class represents any Media Controller graph object identified by an id unique in the media device context. It is subclassed by the Media\+Entity, Media\+Pad and Media\+Link classes that represent the entities, pads and links respectively. They are populated based on the media graph information exposed by the Linux kernel through the MEDIA\+\_\+\+IOC\+\_\+\+G\+\_\+\+TOPOLOGY ioctl.

As the media objects represent their kernel counterpart, information about the properties they expose can be found in the Linux kernel documentation.

All media objects are meant to be created and destroyed solely by the Media\+Device and thus have private constructors and destructors.
