\doxysubsection{Detailed Description}
Base class for camera sensor factories. 

The \doxylink{classlibcamera_1_1CameraSensorFactoryBase}{Camera\+Sensor\+Factory\+Base} class is the base of all specializations of the \doxylink{classlibcamera_1_1CameraSensorFactory}{Camera\+Sensor\+Factory} class template. It implements the factory registration, maintains a registry of factories, and provides access to the registered factories.
