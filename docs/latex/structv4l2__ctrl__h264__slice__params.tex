\doxysubsection{Detailed Description}
struct \doxylink{structv4l2__ctrl__h264__slice__params}{v4l2\+\_\+ctrl\+\_\+h264\+\_\+slice\+\_\+params} -\/ H264 slice parameters

This structure holds the H264 syntax elements that are specified as non-\/invariant for the slices in a given frame.

Slice invariant syntax elements are contained in struct \doxylink{structv4l2__ctrl__h264__decode__params}{v4l2\+\_\+ctrl\+\_\+h264\+\_\+decode\+\_\+params}. This is done to reduce the API surface on frame-\/based decoders, where slice header parsing is done by the hardware.

Slice invariant syntax elements are specified in specification section "{}7.\+4.\+3 Slice header semantics"{}.

Except where noted, the members on this struct match the slice header syntax.
