To facilitate instantiation of \doxylink{classlibcamera_1_1ipa_1_1Algorithm}{Algorithm} classes, the \doxylink{classlibcamera_1_1ipa_1_1AlgorithmFactory}{Algorithm\+Factory} class implements auto-\/registration of algorithms with the IPA \doxylink{classlibcamera_1_1ipa_1_1Module}{Module} class. Each \doxylink{classlibcamera_1_1ipa_1_1Algorithm}{Algorithm} subclass shall register itself using the \doxylink{libipa_2algorithm_8h_a9c0ec8296766fc4ffeba075b5bfce072}{REGISTER\+\_\+\+IPA\+\_\+\+ALGORITHM()} macro, which will create a corresponding instance of an \doxylink{classlibcamera_1_1ipa_1_1AlgorithmFactory}{Algorithm\+Factory} and register it with the IPA \doxylink{classlibcamera_1_1ipa_1_1Module}{Module}.
