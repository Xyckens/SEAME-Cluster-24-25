\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em request} & The request\\
\hline
\end{DoxyParams}
This function is called to instruct the sink to process a request. The sink may process the request synchronously or queue it for asynchronous processing.

When the request is processed synchronously, this function shall return true. The {\itshape request} shall not be accessed by the \doxylink{classFrameSink}{Frame\+Sink} after the function returns.

When the request is processed asynchronously, the \doxylink{classFrameSink}{Frame\+Sink} temporarily takes ownership of the {\itshape request}. The function shall return false, and the \doxylink{classFrameSink}{Frame\+Sink} shall emit the request\+Processed signal when the request processing completes. If the stop() function is called before the request processing completes, it shall release the request synchronously.

\begin{DoxyReturn}{Returns}
True if the request has been processed synchronously, false if processing has been queued 
\end{DoxyReturn}
