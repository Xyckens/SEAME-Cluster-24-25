\doxysubsection{Detailed Description}
IPC message pipe for IPA isolation. 

Virtual class to model an IPC message pipe for use by IPA proxies for IPA isolation. \doxylink{classlibcamera_1_1IPCPipe_a4071e1ef36dfb902b766b5f28330457b}{send\+Sync()} and \doxylink{classlibcamera_1_1IPCPipe_a2c4e67fa7dbf94ccd608cedd0a10dc96}{send\+Async()} must be implemented, and the recv\+Message signal must be emitted whenever new data is available.
