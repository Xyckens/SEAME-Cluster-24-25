\doxysubsection{Detailed Description}
Helper class for dma-\/buf\textquotesingle{}s synchronization. 

This class wraps a userspace dma-\/buf\textquotesingle{}s synchronization process with an object\textquotesingle{}s lifetime.

It\textquotesingle{}s used when the user needs to access a dma-\/buf with CPU, mostly mapped with \doxylink{classlibcamera_1_1MappedFrameBuffer}{Mapped\+Frame\+Buffer}, so that the buffer is synchronized between CPU and ISP.
