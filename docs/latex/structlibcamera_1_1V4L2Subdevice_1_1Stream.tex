\doxysubsection{Detailed Description}
V4\+L2 subdevice stream. 

This class identifies a subdev stream, by bundling the pad number with the stream number. It is used in all stream-\/aware functions of the \doxylink{classlibcamera_1_1V4L2Subdevice}{V4\+L2\+Subdevice} class to identify the stream the functions operate on.
