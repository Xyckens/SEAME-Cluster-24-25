\doxysubsection{Detailed Description}
Describe a range of sizes. 

A \doxylink{classlibcamera_1_1SizeRange}{Size\+Range} describes a range of sizes included in the \mbox{[}min, max\mbox{]} interval for both the width and the height. If the minimum and maximum sizes are identical it represents a single size.

\doxylink{classlibcamera_1_1Size}{Size} ranges may further limit the valid sizes through steps in the horizontal and vertical direction. The step values represent the increase in pixels between two valid width or height values, starting from the minimum. Valid sizes within the range are thus expressed as

width = min.\+width + h\+Step \texorpdfstring{$\ast$}{*} x height = min.\+height + v\+Step \texorpdfstring{$\ast$}{*} y

Where

width \texorpdfstring{$<$}{<}= max.\+width height \texorpdfstring{$<$}{<} max.\+height

Note that the step values are not equivalent to alignments, as the minimum width or height may not be a multiple of the corresponding step.

The step values may be zero when the range describes only minimum and maximum sizes without implying that all, or any, intermediate size is valid. \doxylink{classlibcamera_1_1SizeRange}{Size\+Range} instances the describe a single size have both set values set to 1.
