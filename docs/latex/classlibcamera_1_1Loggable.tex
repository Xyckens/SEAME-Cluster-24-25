\doxysubsection{Detailed Description}
Base class to support log message extensions. 

The \doxylink{classlibcamera_1_1Loggable}{Loggable} class allows classes to extend log messages without any change to the way the \doxylink{libcamera_2base_2log_8h_a3577749fb48d57a158b8ac1a0b3ab57e}{LOG()} macro is invoked. By inheriting from \doxylink{classlibcamera_1_1Loggable}{Loggable} and implementing the \doxylink{classlibcamera_1_1Loggable_a4cc48b0ae7215ceb4bfae6f192b2b3ee}{log\+Prefix()} virtual function, a class can specify extra information to be automatically added to messages logged from class member function.
