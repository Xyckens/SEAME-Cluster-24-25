\doxysubsection{Detailed Description}
unique\+\_\+ptr-\/like wrapper for a file descriptor 

The \doxylink{classlibcamera_1_1UniqueFD}{Unique\+FD} is a wrapper that owns and manages the lifetime of a file descriptor. It is constructed from a numerical file descriptor, and takes over its ownership. The file descriptor is closed when the \doxylink{classlibcamera_1_1UniqueFD}{Unique\+FD} is destroyed, or when it is assigned another file descriptor with \doxylink{classlibcamera_1_1UniqueFD_ac2baf9b8f09783b8207958c55c7b8397}{operator=()} or \doxylink{classlibcamera_1_1UniqueFD_af22d8122d42de7927dbedbdf730d04d0}{reset()}.

Release ownership of the file descriptor without closing it. 

This function releases and returns the owned file descriptor without closing it. The caller owns the returned value and must take care of handling its life time to avoid file descriptor leakages. Upon return this \doxylink{classlibcamera_1_1UniqueFD}{Unique\+FD} is invalid.
