\doxysubsection{Detailed Description}
Base class for camera sensor helper factories. 

The \doxylink{classlibcamera_1_1ipa_1_1CameraSensorHelperFactoryBase}{Camera\+Sensor\+Helper\+Factory\+Base} class is the base of all specializations of the \doxylink{classlibcamera_1_1ipa_1_1CameraSensorHelperFactory}{Camera\+Sensor\+Helper\+Factory} class template. It implements the factory registration, maintains a registry of factories, and provides access to the registered factories.
