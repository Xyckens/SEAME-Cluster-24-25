\doxysubsection{Detailed Description}
A specialized parser for list of key-\/value pairs. 

The \doxylink{classKeyValueParser}{Key\+Value\+Parser} is an options parser for comma-\/separated lists of {\ttfamily key=value} pairs. The supported keys are added to the parser with \doxylink{classKeyValueParser_a5ad0f5f92799b232028f030877570740}{add\+Option()}. A given key can only appear once in the parsed list.

Instances of this class can be passed to the \doxylink{classOptionsParser_ad2a4f00cc2c8f9d3f05c9db1b66f4f06}{Options\+Parser\+::add\+Option()} function to create options that take key-\/value pairs as an option argument. Specialized versions of the key-\/value parser can be created by inheriting from this class, to pre-\/build the options list in the constructor, and to add custom validation by overriding the \doxylink{classKeyValueParser_a53e6ed9e160fe5d5c036ce87318c27b9}{parse()} function.
