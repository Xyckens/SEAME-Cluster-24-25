\doxysubsection{Detailed Description}
Interface for the control validator. 

The \doxylink{classlibcamera_1_1ControlValidator}{Control\+Validator} class is used by the \doxylink{classlibcamera_1_1ControlList}{Control\+List} class to validate controls added to the list. It is an abstract class providing an interface for object-\/specific control validation, such a \doxylink{classlibcamera_1_1Camera}{Camera} controls and V4\+L2 controls.
