\doxysubsection{Detailed Description}
Helper class to create anonymous files. 

Anonymous files behave like regular files, and can be modified, truncated, memory-\/mapped and so on. Unlike regular files, they however live in RAM and don\textquotesingle{}t have permanent backing storage.
