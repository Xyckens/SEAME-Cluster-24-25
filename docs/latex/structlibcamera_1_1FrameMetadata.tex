\doxysubsection{Detailed Description}
Metadata related to a captured frame. 

The \doxylink{structlibcamera_1_1FrameMetadata}{Frame\+Metadata} structure stores all metadata related to a captured frame, as stored in a \doxylink{classlibcamera_1_1FrameBuffer}{Frame\+Buffer}, such as capture status, timestamp and bytes used.
