\doxysubsection{Detailed Description}
A reference to a metadata entry in a buffer.

The data union pointers point to the real data in the buffer, and can be modified in-\/place if the count does not need to change. The count is the number of entries in data of the entry\textquotesingle{}s type, not a count of bytes.
