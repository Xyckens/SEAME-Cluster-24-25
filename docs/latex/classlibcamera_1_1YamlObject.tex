\doxysubsection{Detailed Description}
A class representing the tree structure of the YAML content. 

The \doxylink{classlibcamera_1_1YamlObject}{Yaml\+Object} class represents the tree structure of YAML content. A \doxylink{classlibcamera_1_1YamlObject}{Yaml\+Object} can be empty, a dictionary or list of Yaml\+Objects, or a value if a tree leaf.
