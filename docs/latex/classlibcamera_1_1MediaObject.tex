\doxysubsection{Detailed Description}
Base class for all media objects. 

\doxylink{classlibcamera_1_1MediaObject}{Media\+Object} is an abstract base class for all media objects in the media graph. Each object is identified by a reference to the media device it belongs to and a unique id within that media device. This base class provide helpers to media objects to keep track of these identifiers.
