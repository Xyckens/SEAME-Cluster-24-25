\doxysubsection{Detailed Description}
Helper class that performs computations relating to exposure. 

AEGC algorithms have a need to split exposure between exposure time, analogue and digital gain. Multiple implementations do so based on paired stages of exposure time and gain limits; provide a helper to avoid duplicating the code.
