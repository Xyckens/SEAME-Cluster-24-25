\doxysubsection{Detailed Description}
Base class for pipeline handler factories. 

The \doxylink{classlibcamera_1_1PipelineHandlerFactoryBase}{Pipeline\+Handler\+Factory\+Base} class is the base of all specializations of the \doxylink{classlibcamera_1_1PipelineHandlerFactory}{Pipeline\+Handler\+Factory} class template. It implements the factory registration, maintains a registry of factories, and provides access to the registered factories.
