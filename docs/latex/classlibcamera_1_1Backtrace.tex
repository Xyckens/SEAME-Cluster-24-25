\doxysubsection{Detailed Description}
Representation of a call stack backtrace. 

The \doxylink{classlibcamera_1_1Backtrace}{Backtrace} class represents a function call stack. Constructing an instance captures the call stack at the point the instance is constructed. The instance can later be used to access the call stack and to generate a human-\/readable representation with the \doxylink{classlibcamera_1_1Backtrace_a5a9e4418bd25bd10e3cbe8761801e68e}{to\+String()} function.

Depending on the platform, different backends can be used to generate the backtrace. The \doxylink{classlibcamera_1_1Backtrace}{Backtrace} class provides a best effort to capture accurate backtraces, but doesn\textquotesingle{}t offer any guarantee of a particular backtrace format.

The string representation of the backtrace is a multi-\/line string, with one line per call stack entry. The format of the entries isn\textquotesingle{}t specified and is platform-\/dependent.

The {\itshape skip\+Levels} parameter indicates how many initial entries to skip from the backtrace. This can be used to hide functions that wrap the construction of the \doxylink{classlibcamera_1_1Backtrace}{Backtrace} instance from the call stack. The \doxylink{classlibcamera_1_1Backtrace}{Backtrace} constructor itself is automatically skipped and never shown in the backtrace.

If backtrace generation fails for any reason (usually because the platform doesn\textquotesingle{}t support this feature), an empty string is returned.

A string representation of the backtrace, or an empty string if backtrace generation isn\textquotesingle{}t possible
