Fork and exec a process, and close fds. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em path} & Path to executable \\
\hline
\mbox{\texttt{ in}}  & {\em args} & Arguments to pass to executable (optional) \\
\hline
\mbox{\texttt{ in}}  & {\em fds} & Vector of file descriptors to keep open (optional)\\
\hline
\end{DoxyParams}
Fork a process, and exec the executable specified by path. Prior to exec\textquotesingle{}ing, but after forking, all file descriptors except for those specified in fds will be closed.

All indexes of args will be incremented by 1 before being fed to exec(), so args\mbox{[}0\mbox{]} should not need to be equal to path.

\begin{DoxyReturn}{Returns}
Zero on successful fork, exec, and closing the file descriptors, or a negative error code otherwise 
\end{DoxyReturn}
