The values are signaled on the scale of 0..255. @point\+\_\+cb\+\_\+scaling\+: represents the scaling (output) value for the i-\/th point of the piecewise linear scaling function for cb component. @num\+\_\+cr\+\_\+points\+: specifies represents the number of points for the piece-\/wise linear scaling function of the cr component. @point\+\_\+cr\+\_\+value\+: represents the x coordinate for the i-\/th point of the piece-\/wise linear scaling function for cr component.
