@width\+: Grid horizontal dimensions, in number of grid blocks(cells). For AWB, the range is (16, 80). For AF/\+AE, the range is \mbox{[}3, 7\mbox{]}. @height\+\_\+per\+\_\+slice\+: The number of blocks in vertical axis per slice. Default 2. @x\+\_\+start\+: X value of top left corner of Region of Interest(\+ROI). @y\+\_\+start\+: Y value of top left corner of ROI @x\+\_\+end\+: X value of bottom right corner of ROI @y\+\_\+end\+: Y value of bottom right corner of ROI

Due to the size of total amount of collected data, most statistics create a grid-\/based output, and the data is then divided into "{}slices"{}.
