\doxysubsection{Detailed Description}
A camera lens based on V4\+L2 subdevices. 

The \doxylink{classlibcamera_1_1CameraLens}{Camera\+Lens} class eases handling of lens for pipeline handlers by hiding the details of the V4\+L2 subdevice kernel API and caching lens information.
