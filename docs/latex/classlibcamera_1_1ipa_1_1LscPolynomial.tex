\doxysubsection{Detailed Description}
Class for handling even polynomials used in lens shading correction. 

Shading artifacts of camera lenses can be modeled using even radial polynomials. This class implements a polynomial with 5 coefficients which follows the definition of the Fix\+Vignette\+Radial opcode in the Adobe DNG specification.
