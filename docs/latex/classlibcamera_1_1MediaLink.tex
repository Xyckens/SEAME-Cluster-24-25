\doxysubsection{Detailed Description}
The \doxylink{classlibcamera_1_1MediaLink}{Media\+Link} represents a link between two pads in the media graph. 

Links are created from the information provided by the Media Controller API in the \doxylink{structmedia__v2__link}{media\+\_\+v2\+\_\+link} structure. They reference the \doxylink{classlibcamera_1_1MediaLink_aaf97df0f7ee1b64bfaa52816fafe14e0}{source()} and \doxylink{classlibcamera_1_1MediaLink_a9831d560d360e6fbb3b01e35455fcedd}{sink()} pads they connect and track the link status through link flags().

Each link is referenced in the link array of both of the pads it connect.

Enable or disable a link. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em enable} & True to enable the link, false to disable it\\
\hline
\end{DoxyParams}
Set the status of a link according to the value of {\itshape enable}. Links between two pads can be set to the enabled or disabled state freely, unless they\textquotesingle{}re immutable links, whose status cannot be changed. Enabling an immutable link is not considered an error, while trying to disable it is.

Enabling a link establishes a data connection between two pads, while disabling it interrupts that connection.

\begin{DoxyReturn}{Returns}
0 on success or a negative error code otherwise
